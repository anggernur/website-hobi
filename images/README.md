# Petunjuk Menambahkan Gambar

Folder `images/` digunakan untuk menyimpan semua gambar website.

## File Gambar yang Diperlukan

Berikut file gambar yang harus ditambahkan ke folder ini:

### Foto Profil
- **skater.jpg** - Foto Anda sambil skateboarding (recommended: 600x600px atau lebih)

### Gallery Images
- **skate1.jpg** - Foto skating di park (recommended: 400x300px)
- **skate2.jpg** - Foto performing tricks (recommended: 400x300px)
- **skate3.jpg** - Foto street skating (recommended: 400x300px)
- **skate4.jpg** - Foto bersama komunitas (recommended: 400x300px)
- **skate5.jpg** - Foto kompetisi skateboard (recommended: 400x300px)
- **skate6.jpg** - Foto koleksi board (recommended: 400x300px)

## Cara Menambahkan Gambar

1. **Siapkan gambar Anda** (format JPG, PNG, atau WebP)
2. **Rename** sesuai nama file yang diperlukan di atas
3. **Copy** ke folder `images/`
4. **Refresh** browser untuk melihat gambar

## Catatan Penting

- Jika gambar tidak tersedia, website akan otomatis menampilkan **placeholder SVG** berwarna
- Ini dilakukan agar website tetap berfungsi baik bahkan tanpa gambar
- File placeholder auto-generate di browser menggunakan JavaScript

## Format & Ukuran Optimal

| Gambar | Format | Ukuran | Notes |
|--------|--------|--------|-------|
| Foto Profil | JPG/PNG | 600x600px | Untuk section #tentang |
| Gallery | JPG/PNG | 400x300px | Untuk section #galeri |
| Logo/Icon | PNG/SVG | Sesuai kebutuhan | Opsional |

## Tips Mengedit Gambar

Gunakan tools online gratis:
- **Canva** (canva.com) - Design graphic
- **Pixlr** (pixlr.com) - Edit foto online
- **Unsplash** (unsplash.com) - Download foto gratis
- **Pexels** (pexels.com) - Foto berkualitas tinggi gratis
- **Skateboard themed images** - Cari di Google Images dengan license "Creative Commons"

## Compress Gambar

Untuk performa optimal, compress gambar Anda:
- **TinyPNG** (tinypng.com) - Compress JPG & PNG
- **ImageOptim** - Tool compress offline
- Target: < 200KB per gambar

## Troubleshooting

**Gambar tidak muncul?**
1. Cek nama file sesuai dengan HTML
2. Cek path: `images/nama-file.jpg`
3. Cek format & encoding gambar
4. Refresh browser (Ctrl+F5)

**Gambar terlalu besar?**
1. Resize ke ukuran yang sesuai
2. Compress menggunakan TinyPNG
3. Convert ke format WebP untuk file lebih kecil

---

Website ini dilengkapi dengan **automatic fallback SVG placeholder**, jadi Anda bisa test website tanpa menambahkan gambar terlebih dahulu!
